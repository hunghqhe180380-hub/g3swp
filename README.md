# POET - Java Web Project

## Development environment
- JDK 8+
- Apache Tomcat 10
- Port: 8080
- URL: http://localhost:8080/POET

## Tech Stack
- Java Servlet (Jakarta EE)
- JSP + JSTL
- SQL SSMS Server
- NetBeans
- Tomcat 10

## Features
- Login / Logout
- Register
- Session-based authentication

## How to run
1. Clone repo
2. Open NetBeans
3. Run project
